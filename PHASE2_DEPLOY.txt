import { useState } from "react";
import SmartIntake from "./components/SmartIntake";
import AgentDashboard from "./components/AgentDashboard";
export default function App() {
 const [agentData, setAgentData] = useState(null);
 if (agentData) {
   return (
     <AgentDashboard
       agentData={agentData}
       onBack={() => setAgentData(null)}
     />
   );
 }
 return <SmartIntake onComplete={(data) => setAgentData(data)} />;
}
IMPORTANT: SmartIntake.jsx must call onComplete(data) when the user
finishes the intake and clicks "BUILD MY BLUEPRINT". Verify this prop
is wired in SmartIntake.jsx before deploying. The data shape passed
to onComplete should include at minimum: agentName, concept, inputs,
outputs, triggers. AgentDashboard handles both v5 and v6 blueprint formats.

=== WHAT IS IN AGENTDASHBOARD.JSX ===
LaunchScreen: 4-phase deploy animation. Phase 1 calls Claude API to
generate an optimized execution system prompt from the blueprint JSON.
Phases 0, 2, 3 are timed delays. On completion, transitions to live dashboard.
AgentDashboard (main):
- Header: agent name, LIVE badge, back button, run counter
- Stats bar: total runs, success rate, avg time, avg cost (shows after first run)
- Tabs: RUN / HISTORY / IMPROVE
RUN tab:
- File drop zone + paste textarea
- Run button (calls Claude API with generated system prompt + user input)
- Structured output display: DETECTED, EXTRACTED, ANALYSIS, OUTPUT, FLAGS
- Download output as JSON
- Stripe gate after FREE_RUN_LIMIT runs (default: 1)
HISTORY tab:
- All runs as expandable cards (newest first)
- Each card: timestamp, duration, cost estimate, flag count
- Expand for full structured output
IMPROVE tab:
- Agent definition summary (blueprint fields)
- Improvement queue: 3 suggestions generated by Claude from run patterns
- Each suggestion: title, plain English description, action button
StripeGateModal: blocks run after free limit, shows $199/month offer.
