FIXES IN THIS BUILD:
=============================================

1. MULTIPLE HINT SELECTIONS + UNDO
   - Click an option → adds it AND keeps remaining options visible
   - Shows "ADD MORE:" label with remaining options still clickable
   - Each added item has an "undo" link next to it
   - Can add all 3 options or any combination

2. BLUEPRINT ALWAYS SHOWS ON COMPLETION
   - "BUILD MY BLUEPRINT" always shows blueprint screen first
   - Blueprint screen has "START ACADEMY → DEPLOY & IMPROVE" button
   - That button then hands off to the Academy
   - User never skips past the blueprint again

3. SYSTEMS STEP — SIMPLEST FIRST ENFORCED
   - Coach prompt now explicitly says: Box/Google Drive/shared folder FIRST
   - Only suggest Procore, Salesforce, Slack etc. if user mentioned them
   - No more complex API suggestions by default

4. DOCUMENT LIBRARY MESSAGING
   - Upload panels now say "uploaded once — lives in agent's permanent
     document library — available every run automatically"

5. ALL PLACEHOLDER TEXT NOW GENERIC
   - No construction-specific examples in placeholders
   - Zero hardcoded industry specifics outside getDomainContext()

6. TEMPLATE FIELD CATEGORIZATION
   - Template upload now categorizes fields into 3 buckets:
     CYAN = fields extracted from source document (qty, items, prices)
     GOLD = fields user provides manually (job number, codes, approver)
     GREY = fields agent computes (totals, page numbers)
   - All subsequent coach prompts use these categories for accuracy

APP.JSX INTEGRATION:
=============================================

Import both components:

  import PreviewSandbox from './components/PreviewSandbox'

  const [currentStep, setCurrentStep] = useState(0)

  <div style={{
    display: "flex",
    flexDirection: window.innerWidth < 768 ? "column" : "row",

    <div style={{ flex: window.innerWidth < 768 ? "none" : "0 0 60%" }}>
      <SmartIntake onComplete={(data) => setAgentData(data)} />

      minHeight: window.innerWidth < 768 ? "500px" : "100vh"
    }}>
      <PreviewSandbox agentData={agentData} currentStep={currentStep} />

  </div>
